Instruções de Instalação

A maneira mais recomendada de instalar o pacote é via Composer. Obtê-lo é tão simples quanto executar o seguinte comando: composer require claudsonm/cep-promise-php.

Após instalar você precisará importar o autoloader do Composer em seus scripts PHP para ter acesso às dependências instaladas através dele. Para isso adicione require 'vendor/autoload.php'; nos arquivos PHP nos quais deseja utilizar qualquer uma das dependências.
